\section*{Echo}

Entwickle ein Programm welches einen echo-Befehl mit Platzhaltern unterstüzt.


\begin{description}
 \item[set $<$variable$>$ $<$value$>$] 
Setze die Variable auf den gewünschten Wert
 \item[show] Liste alle Variablen und Werte auf.
  \item[echo $<$text$>$] Gib den angegebenen Text aus. Hier sind die Variablen einzusetzen. (siehe Beispiel)
\end{description}

 

\subsection*{Beispiel}
\begin{verbatim}
> set name Christoph
> set job Tutor
> set group 6
show
name: Christoph
job: Tutor
group: 6
> echo $name ist $job der Gruppe $group
Christoph ist Tutor der Gruppe 6
\end{verbatim}

\subsection*{Aufruf}
\texttt{./echo}
